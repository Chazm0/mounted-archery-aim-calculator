# T90 骑射预瞄提前量计算器（Streamlit 网页版）

这是一个简化版 T90 骑射预瞄计算工具。

## 运行方式

先安装依赖：

```bash
pip install -r requirements.txt
```

运行网页：

```bash
streamlit run app.py
```

然后浏览器会自动打开本地网页。

## 模型假设

- 计时跑道默认 90 m。
- 三个靶心默认在计时跑道中点 45 m。
- 骑手位置直接等同为马的位置。
- 默认横向距离为 10 m，即规则中的跑道内侧到靶心 9 m + 马在跑道中央约 1 m。
- 靶面角度使用“靶面平面相对跑道方向的角度”。
- 不计算风、空气阻力、撒放延迟、马加速度、拉距变化。

## 箭速来源

支持三种：

1. 直接输入实测箭速。
2. 用弓磅数 + 箭重估算。
3. 用步射上瞄/下坠数据估算有效箭速。

## 输出

- 箭飞行时间
- 沿跑道等效提前量
- 靶面等效偏移
- 从靶心向起点侧几个靶径
- 超过靶边几个靶径
- 垂直向上补偿几个靶径
